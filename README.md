# DigiOne Bank
![Kotlin](https://img.shields.io/badge/Kotlin-%230095D5.svg?&logo=kotlin&logoColor=white)

DigiOne Bank é uma aplicação de serviços bancários desenvolvida durante o bootcamp Everis Kotlin Developer da Digital Innovation One.

---


## ✍️ Aprendizados

#### Criando o projeto (08/03/2021):
> Na primeira aula criei o projeto de uma forma diferente da que geralmente faço. Clonei o repositório do professor [@jether2011](https://github.com/jether2011) com a opção de vcs no IntelliJ, como foi mostrado.
>
> O fato engraçado desta aula (e do curso inteiro) foi que aprendi mais sobre Git do que sobre Kotlin. Quando clonei o repositório, veio todo o código do projeto que já tinha sido desenvolvido, mas pretendia começar do zero enquanto acompanhava as aulas. Então resolvi passear pelo fórum da DIO e encontrei um post do [@clovisdanielcosta](https://github.com/clovisdanielcosta) ensinando [como iniciar o projeto vazio](https://cutt.ly/az2qXAI).
#### Adicionando a primeira classe (09/03/2021):
> Ao criar e testar a classe Pessoa, que foi a primeira adicionada ao projeto, tentei fazer um pouco diferente do que foi mostrado e já fui declarando as properties no construtor da classe porque imaginei que elas ficariam assim mais à frente no projeto.
>
> Já venho estudando Kotlin há alguns meses e sempre acho divertido criar uma classe sem me perder no meio de um monte de palavras, como acontecia quando comecei a aprender Java na graduação. So simple and cute.
#### Data Class (10/03/2021):
> Criei algumas Data Classes antes e sabia como funcionava, mas gostei muito de ter um exemplo no curso para poder praticar mais. 
>
> No projeto, foi criada a Data Class "Banco", que tinha uma função para retornar informações (nome e número) de um banco.
#### A Enum Class (11/03/2021):
> Sobre Enum Class > já tinha visto o nome em algum lugar, mas nunca tinha estudado nada sobre. Então criei minha primeira Enum Class neste projeto. Achei bem legal. Muito simples de entender e de utilizar.
